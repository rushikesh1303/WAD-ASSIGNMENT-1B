# Web Application Development: Assignment 1(B)

## Problem Statement
Create a JavaScript-based User Registration system that captures data from a form and stores it using:
1. **Local Storage** For persistent data handling.
2. **AJAX POST Method:** To simulate or perform data submission.
3. **Data Display:** Show the list of registered users on a separate page.

## Technologies Used
- **HTML5:** For the registration form structure.
- **CSS3 / Bootstrap:** For styling and responsiveness.
- **JavaScript (ES6):** For data handling and logic.
- **AJAX (XMLHttpRequest/Fetch):** For the POST method implementation.
